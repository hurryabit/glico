# glico — GitHub PR Link Copier

A Chrome extension that copies the current GitHub pull request as a formatted link to the clipboard. Rich text targets (Notion, Google Docs, email) receive a hyperlink; plain text targets receive a Markdown link.

## Installation

1. Clone this repository.
2. Open `chrome://extensions` in Chrome.
3. Enable **Developer mode** (toggle in the top-right corner).
4. Click **Load unpacked** and select the cloned directory.

Pin the extension to your toolbar for quick access.

## Usage

Navigate to any GitHub pull request and click the extension icon. The link is copied instantly.
